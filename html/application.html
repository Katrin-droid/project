Login page

In the login page you can see two fields and a line of text that its underlined.
In order to login you have to have an account that you can create.
When you press the underlined line of text you will see a new page opening for you.
In this page you can create an account.

Register page

In the register page you can see five different fields
that you will use to create an account.
If you already have an account or you just made an account,
just press the underlined line of text below the button register.

Options page

In the options page you can see three different buttons
(Calendar,Complaints,Rules)every button leads to a different page.
In the top left corner you can see another button(logout).

Calendar page

In the calendar page you can add events,
you can see when an employee of the agency will visit the house.
The purpouse of this page is to inform housemates about an upcoming party that
you want to host at a specific date and time.

Complaints page

In the complaits page you can see three buttons
that allow the tenants to issue a complaint,
to remove the complaint if they made a mistake and to show complaints in the box.

Rules page

This page with rules is created
because we want every tenant to know what rules he has to follow.
The rules can be changed if the admin decides to do so.
